Multicast API

Detailed Description

API functions listed to create/delete multicast trees and to add/delete multicast routes MFIB

The multicast API's are controlled based on the multicast protocols that propagates the route entries to MFIB. A multicast tree handle is programmed against each route in mfib. The tree is derived in the queuing block and the packet is replicated based on the members.
