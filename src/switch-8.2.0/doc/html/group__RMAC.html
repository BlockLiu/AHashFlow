Typedef Documentation

typedef enum switch_rmac_type_s switch_rmac_type_t

Router mac type.

Outer or inner rmac table is programmed based on the rmac type. source mac rewrite index is allocated only for the first mac entry. smac index is not reallocated or reprogrammed when the first mac is deleted.
